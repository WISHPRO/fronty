Fronty

Fronty is a boilerplate for starting new front-end projects

  • Download
  • View on GitHub

Sass

Fronty is using Compass to boost sass files. It has only helpers (mixins, variables, etc) without any UI component.

JavaScript

Like sass, JavaScript code also doesn't have anything but basic structure. Files are compiled via browserify.

Build

Gulp is main tool for compiling files in Fronty. All files (css, javascript, images) are watched when you run gulp watch.
